链表分为单向链表和双向链表：
    单向链表：当前节点指向下一个节点
    双向链表：当前节点的头指向上一个节点的尾，尾指向下一个节点的头。
还可以分为直线链表和环形链表：
    直线链表：有最终头结点和尾结点
    环形链表：没有最终头结点和尾结点

我以为的链表是这个亚子滴：
    01 data 02 -> 02 data 03 -> 03 data 04 -> 04 data 05 -> 05 data 01 ->
实际上是这个亚子滴：
    04 data 05 ->
    01 data 02 ->
    05 data 01 ->
    03 data 04 ->
    02 data 03 ->
