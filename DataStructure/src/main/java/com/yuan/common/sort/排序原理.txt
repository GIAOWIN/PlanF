选择排序：
    名称:
        SelectSort
    思路：
        第一次从arr[0+0]~arr[arr.length-1]中找出最小的数并与arr[0+0]交换
        第二次从arr[0+1]~arr[arr.length-1]中找出最小的数并与arr[0+1]交换
        第二次从arr[0+2]~arr[arr.length-1]中找出最小的数并与arr[0+2]交换
        ...
插入排序：
    名InsertionSort
    思路：
        n个待排序元素 看成两张表
        一张只有一个元素有序表
        一张n-1个元素的无序表
        把无序表中的每个元素依次取出有序的插入有序表中
        ...
